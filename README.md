=======================

エウレカコンピュータの試験問題

-----
http://www.eurekacomputer.jp/recruit.html
文字、文字列、数値リテラルを一切使わず、画面上に”Hello World\n”を表示しなさい。ソースは300文字以内とします
（改行は1文字と数えます）

圧縮したものは３００字未満

##### 動作確認方法
 min.txtが最小化版
 processingのIDEを起動して、スケッチをJavascriptにする。  
 min.txtの内容をコピペして実行
